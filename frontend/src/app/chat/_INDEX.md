# frontend/src/app/chat 架构说明

用户端智能体对话界面
⚠️ 文件夹变化时请更新此文件

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| page.tsx | 核心 | 用户对话主页面 |

## 功能说明

用户端对话界面，供普通员工使用：
- 左侧：智能体选择列表（常用/全部）
- 中间：多轮对话区域
- 右侧抽屉：历史对话记录

## 路由

```
/chat   # 用户对话界面（独立布局，不使用 AdminLayout）
```

## 依赖

- 不依赖 AdminLayout，使用独立的 Layout 布局
- 智能体数据与 /admin/agents 共享同一后端 API
